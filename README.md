# Performance of an M-Cycle-Based Indirect Evaporative Cooling System

## Overview

This repository contains the Python source code and supporting materials associated with the research article:

**"Parametric Analysis of Air Velocity Impact on the Performance of an M-Cycle-Based Indirect Evaporative Cooling System"**

The code was developed to investigate the influence of inlet air velocity on the thermodynamic, hydraulic, and energy performance of a Maisotsenko Cycle (M-Cycle) based indirect evaporative cooling system.

## Repository Contents

- `main.py` – Main simulation code for thermal and fluid-flow analysis of the M-Cycle heat and mass exchanger.
- `requirements.txt` – List of required Python packages.

## Requirements

- Python 3.10 or later

Required packages:

- numpy
- matplotlib

Install dependencies using:

```bash
pip install -r requirements.txt
```

## Running the Code

Execute the simulation using:

```bash
python main.py
```

The program calculates and/or generates:

- Outlet air temperature
- Relative humidity
- Cooling capacity
- Convective heat transfer coefficients
- Number of Transfer Units (NTU)
- Fan power consumption
- Coefficient of Performance (COP)
- Energy Efficiency Ratio (EER)
- Seasonal Energy Efficiency Ratio (SEER)
- Pressure losses
- Water consumption estimates
