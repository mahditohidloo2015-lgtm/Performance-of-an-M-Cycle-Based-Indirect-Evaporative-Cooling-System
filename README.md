# Dataset and Code Repository

This repository contains the base simulation code and supporting materials used in the research study.

## Contents

### Code
- `main.py`
  Main simulation model for thermal and fluid analysis of the heat and mass exchanger system.

### Requirements
Python 3.10+

Required Python packages:
- numpy
- matplotlib

Install dependencies using:

pip install -r requirements.txt

## Running the Code

Run the simulation using:

python main.py

The code generates:
- Temperature distribution plots
- Thermal analysis results
- Cooling performance indicators
- COP, EER, NTU, pressure losses, and water usage calculations
